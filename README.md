# 📈 StockVision AI - AI-Powered Stock Market Forecasting

![Python](https://img.shields.io/badge/Python-3.11-blue?logo=python)
![React](https://img.shields.io/badge/React-Frontend-61DAFB?logo=react)
![FastAPI](https://img.shields.io/badge/FastAPI-Backend-009688?logo=fastapi)

![License](https://img.shields.io/badge/License-MIT-green)

An AI-powered stock market forecasting platform that combines **Transformer and LSTM deep learning models**, **technical indicators**, and **financial news sentiment analysis** to predict stock price trends. The application provides an interactive dashboard for visualizing historical data, AI-generated forecasts, and market insights.

---

## 🚀 Features

* 📈 Stock price forecasting using Transformer & LSTM models
* 🤖 AI-powered prediction engine
* 📰 Financial news sentiment analysis
* 📊 Interactive stock market dashboard
* 📉 Historical stock price visualization
* ⚡ FastAPI REST API backend
* 💻 Responsive React frontend
* 📂 Modular project architecture

# ⚙️ Getting Started

## Clone the Repository

```bash
git clone https://github.com/your-username/StockVision-AI.git
cd StockVision-AI
```

---

## Backend Setup

```bash
cd backend
pip install -r requirements.txt
python main.py
```

The backend will start on:

```text
http://localhost:8000
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

The frontend will start on:

```text
http://localhost:5173
```
